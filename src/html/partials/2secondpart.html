VARIABLES

Variable means anything that can vary. In JavaScript, a variable stores the data value that can be changed later on. Use the reserved keyword LET or CONST to declare a variable in JavaScript.

Conditional statements control behavior in JavaScript and determine whether or not pieces of code can run. There are multiple different types of conditionals in JavaScript.

A JavaScript function is a block of code designed to perform a particular task. A JavaScript function is executed when "something" invokes it (calls it).

JavaScript's interaction with HTML is handled through events that occur when the user or the browser manipulates a page. When the user clicks a button, that click is an event.
